# openshift-hybrid-install

Installation of the OpenShift Container Platform 4 on VMware infrastructure for the `master` and `infra` nodes and HPE Synergy BareMetal servers for `compute`.

## Pre-requisites

- DNS Server
- DHCP Server
- NTP Server
- HAProxy Server
